# Handler Example

간단히 핸들러를 생성하여 반복 작업과 화면으로의 처리 방법을 익히기 위한 예제.

# 중요 학습 내용

handler에서 post()를 통한 재귀적 반복 작업 학습

thread에서 직접적으로 UI 변경 -> exception 발생
  - 을 방지하기 위해 handler에게 UI 변경 명령 할당.
