# ContextMenu Example

리스트에, 리스트마다 특색있는 context(꾹 누르면 나옴) 메뉴를 적용해보는 예제.

# How to make?

1. res에 메뉴 구현
2. registerForContextMenu를 통해 해당 뷰 등록
3. onCreateContextMenu를 통해 등록된 id별로 원하는 contextView 호출.
