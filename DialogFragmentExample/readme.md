# DialogFragment Example
  - DialogFragment를 직접 구현해보고, 사용 방법을 학습한다.

# 주요 학습 내용
  - Alert Dialog를 Fragment로 만들어 사용할 수 있도록 제공되는 Fragment.
  - 기능 상 Alert Dialog와 큰 차이는 없음.
  - Dialog를 show 할 때는 `dialog.show(supportFragmentManager, "tag")`의 형식으로, 생성자에 매니저를 필요로 한다.
