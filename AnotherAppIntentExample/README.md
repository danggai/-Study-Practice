# AnotherAppIntent Example

다른 어플리케이션으로의 Intent를 실습해보는 예제.

# 중요 학습 내용

1. intent할 대상을 외부 어플리케이션으로 정해두는 것으로 자신의 어플리케이션으로 이동 가능.
    - 대상과 같은 id를 가진 어플리케이션이 중복으로 존재할 때, 사용자에게 선택지가 주어진다.

2. 안드로이드 내장 어플리케이션중 일부는 이미 사용할 수 있도록 제공 되어 있음.
