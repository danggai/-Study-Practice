# ListFragment Exmaple

  - List로 이루어진 Fragment를 구현해보는 예제.

# 주요 학습 내용

  - Fragment 내에 ListView를 사용할 경우 보다 편리하게 구성 할 수 있음.
  - ListView의 id가 @android:id/list로 설정되면 자동으로 ListView를 찾아 관리함.
